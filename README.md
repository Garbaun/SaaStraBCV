# Saastra 🚀

> Micro-SaaS fikirleri ve girişimcilik ilhamı için retro-fütüristik bir platform

![Saastra Preview](https://trae-api-sg.mchost.guru/api/ide/v1/text_to_image?prompt=retro%20crt%20monitor%20display%20with%20pixelated%20text%20showing%20micro%20saas%20ideas%2C%20neon%20green%20colors%2C%20scanlines%2C%2080s%20aesthetic%2C%20dark%20background&image_size=landscape_16_9)

## 🎯 Proje Hakkında

Saastra, girişimciler ve geliştiriciler için 126+ Micro-SaaS fikri sunan, retro bilgisayar estetiğiyle tasarlanmış bir ilham platformudur. CRT monitör görünümü ve pixel-art tarzıyla klasik bilgisayar dönemini anımsatan modern bir web uygulamasıdır.

## ✨ Özellikler

- 🖥️ **Retro CRT Monitor Tasarımı** - Özel Commodore CRT görseli ile gerçekçi retro deneyimi
- 📱 **Responsive Tasarım** - Tüm cihazlarda mükemmel görünüm
- 🎮 **Smooth Scroll Animasyonu** - "GAME OVER" butonu ile yavaşça yukarı kaydırma
- 🧠 **Random Fikir Üretici** - 126+ Micro-SaaS fikri arasından rastgele seçim
- ⚡ **Thinking State UX** - Profesyonel 2 saniyelik düşünme animasyonu
- 🎨 **Pixel-Art Görselleştirme** - Scanlines, titreme efektleri ve CRT parıltıları

## 🛠️ Teknoloji Stack

- **Frontend:** React 18.3.1 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Icons:** Lucide React
- **Deployment:** GitHub Pages (Ready)

## 🚀 Hızlı Başlangıç

### Gereksinimler

- Node.js (v18 veya üzeri)
- npm veya pnpm

### Kurulum

```bash
# Projeyi klonlayın
git clone https://github.com/Garbaun/saastra.git

# Proje dizinine girin
cd saastra

# Bağımlılıkları yükleyin
npm install
# veya
pnpm install

# Geliştirme sunucusunu başlatın
npm run dev
# veya
pnpm dev
```

### Production Build

```bash
# Uygulamayı derleyin
npm run build
# veya
pnpm build

# Önizleme için sunucu başlatın
npm run preview
# veya
pnpm preview
```

## 🎮 Bileşenler

### HeroSection
- Özel Commodore CRT monitor görseli

- CRT efektleri (scanlines, glow, flicker)

### StatsSection
- Micro-SaaS istatistikleri
- "GAME OVER? NO! PRESS START!" butonu
- Özel smooth scroll animasyonu (2 saniye)

- 126+ Micro-SaaS fikri veritabanı
